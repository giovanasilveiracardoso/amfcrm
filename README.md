# Configuração do projeto #

## Pré-requisitos ##

Java 17, Maven e Docker instalado.

## Baixe o projeto com o comando abaixo: ##

$ git clone https://github.com/giovanasilveiracardoso/amfcrm.git

## Faça o build do projeto: ##

$ mvn clean

$ mvn package -Pproduction

## Suba o projeto: ##

$ docker-compose up
